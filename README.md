# YouTube Telegram Bot

Telegram бот для поиска и скачивания видео/аудио с YouTube через yt-dlp.

## Возможности

- 🔍 Поиск видео по запросу (команда или текстовое сообщение)
- 🔥 Популярные видео (trending)
- ⭐ Персональные рекомендации YouTube (через cookies.txt)
- 🎬 Скачивание видео в MP4 (до 480p)
- 🎵 Скачивание аудио в MP3
- 📦 Автоматическая разбивка больших файлов на части
- 🔗 Прямое скачивание по ссылке YouTube
- 🍪 Управление cookies через бота

## Требования

- **Node.js** 18+
- **yt-dlp** — `winget install yt-dlp`
- **ffmpeg** — `winget install ffmpeg`

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env`:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
MAX_FILE_SIZE_MB=50
VIDEO_QUALITY=480p
```

3. Получите токен бота у [@BotFather](https://t.me/BotFather)

## Запуск

```bash
npm start
```

Для разработки с автоперезагрузкой:
```bash
npm run dev
```

## Использование

| Команда / Действие | Описание |
|---|---|
| `/start` | Приветствие и инструкции |
| `/search <запрос>` | Поиск видео на YouTube |
| `/trending` | Популярные видео |
| `/recommendations` | Персональные рекомендации (нужен cookies.txt) |
| `/cookies` | Управление cookies |
| `/help` | Помощь |
| Текстовое сообщение | Автоматический поиск |
| Ссылка YouTube | Прямое скачивание |
| Файл cookies.txt | Загрузка cookies для рекомендаций |

## Cookies и рекомендации

Для доступа к персональным рекомендациям YouTube:

1. Установите расширение браузера **"Get cookies.txt LOCALLY"**
2. Зайдите на youtube.com и авторизуйтесь
3. Экспортируйте cookies через расширение
4. Отправьте файл `cookies.txt` боту в Telegram

## Структура проекта

```
├── src/
│   ├── index.js       # Основной файл бота
│   ├── youtube.js     # Поиск и рекомендации через yt-dlp
│   ├── downloader.js  # Скачивание видео/аудио
│   └── utils.js       # Утилиты, cookies, проверки
├── downloads/         # Временные файлы (создается автоматически)
├── cookies/           # Cookies пользователей (создается автоматически)
├── .env               # Конфигурация
└── package.json       # Зависимости
```

## Примечания

- Видео скачиваются во временную папку и удаляются после отправки
- Файлы > 50 МБ автоматически разбиваются на части
- Cookies хранятся в папке `cookies/` отдельно для каждого пользователя
- Для работы необходимы системные утилиты yt-dlp и ffmpeg
